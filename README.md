# 개인정보처리방침

**푸드트럭 매니저**(이하 "앱")는 사용자의 개인정보를 중요하게 생각하며, 개인정보 보호 관련 법령을 준수합니다.

## 1. 수집하는 정보

본 앱은 회원가입 및 로그인 기능을 제공하지 않으며, 사용자의 개인정보를 수집하거나 서버에 저장하지 않습니다.

앱에서 입력한 데이터(매출, 비용, 행사 일정, 서류 정보 등)는 사용자의 기기에만 저장됩니다.

## 2. 저장되는 정보

앱은 다음 정보를 사용자의 기기에 저장할 수 있습니다.

- 매출 정보
- 비용 정보
- 순수익 정보
- 행사 일정
- 행사 지원 현황
- 서류 정보
- 백업 데이터(JSON)

모든 정보는 사용자 기기 내부에만 저장됩니다.

## 3. 파일 접근

앱은 사용자가 선택한 경우에만 파일을 선택하거나 공유하기 위해 저장소 접근 권한을 사용할 수 있습니다.

선택한 파일은 사용자의 요청이 있을 때만 공유되며, 외부 서버로 전송되지 않습니다.

## 4. 인터넷 통신

본 앱은 사용자 데이터를 개발자의 서버로 전송하지 않습니다.

## 5. 광고

본 앱은 광고를 포함하지 않습니다.

## 6. 제3자 제공

본 앱은 어떠한 개인정보도 제3자에게 제공하지 않습니다.

## 7. 개인정보 보호

모든 데이터는 사용자의 기기에만 저장되며 개발자는 해당 정보를 수집하거나 열람할 수 없습니다.
